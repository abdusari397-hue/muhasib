# تطبيق محاسب (Muhasib) 💰
> تطبيق ويب ذكي وراقٍ لإدارة ديونك الشخصية ومستحقاتك المالية بأسلوب عصري وسلس.

تطبيق ويب مصمم بمفهوم "الهاتف أولاً" (Mobile-First) باستخدام تصميم زجاجي عصري (Glassmorphism)، يدعم اللغة العربية بشكل كامل (RTL)، مع تجربة مستخدم تفاعلية وحفظ تلقائي للبيانات.

---

## 🌟 الميزات الرئيسية (Features)

1. **المدخل الذكي الديناميكي (Smart Wizard Inputs)**:
   - بمجرد إدخال المبلغ وتحديد نوع المعاملة، يتغير النموذج تلقائياً:
     - إذا كانت المعاملة **لك (حقوقك)** 🟩: يطلب منك النظام الاسم تحت عنوان **"من من؟"** (المدين).
     - إذا كانت المعاملة **عليك (ديونك)** 🟥: يطلب منك النظام الاسم تحت عنوان **"لمن؟"** (الدائن).
2. **شاشة ملخص البيانات (Interactive Dashboard)**:
   - احتساب فوري وإجمالي للمبالغ التي لك والتي عليك.
   - حساب **صافي الرصيد المعلق** وتغيير لون واجهة الرصيد تلقائياً (أخضر للربح/الفائض، أحمر للدين/العجز).
3. **تصفية وبحث لحظي (Live Search & Filter)**:
   - البحث عن المعاملات بالاسم أو الملاحظات أثناء الكتابة.
   - تصفية سريعة بنقرة واحدة (الكل، لك، عليك).
4. **تأثيرات حركية خفيفة (Micro-animations)**:
   - حركات انتقال سلسة عند التغيير بين الخيارات أو مسح المعاملات.
5. **حفظ محلي فوري (Local Storage Persistence)**:
   - تُخزن جميع معاملاتك على متصفحك تلقائياً دون الحاجة لقاعدة بيانات خارجية.

## 🚀 كيفية تشغيل التطبيق (How to Run)

بما أن التطبيق مبني بلغات الويب الأساسية (HTML5 / CSS3 / JavaScript)، يمكنك تشغيله مباشرة بعدة طرق:

### الطريقة الأولى: التشغيل المباشر
- قم بفتح ملف `index.html` في متصفح الويب الخاص بك مباشرة (عبر النقر المزدوج).

### الطريقة الثانية: عبر خادم محلي (موصى به)
إذا كنت تفضل تشغيل التطبيق عبر خادم محلي (Server)، يمكنك استخدام الخادم المفعل حالياً على جهازك:
1. افتح المتصفح واذهب إلى العنوان التالي:
   **[http://localhost:8080](http://localhost:8080)**

أو قم بتشغيل أي خادم آخر من سطر الأوامر في مجلد المشروع:
- باستخدام بايثون:
  ```bash
  python -m http.server 8080
  ```
- باستخدام Node.js:
  ```bash
  npx http-server -p 8080
  ```

## 📂 هيكلة الملفات (File Structure)

- `index.html` - الهيكل البنائي وعناصر الواجهة والمكتبات (Lucide Icons, Google Fonts).
- `styles.css` - نظام التصميم والألوان (HSL)، الأنماط الزجاجية، والتجاوب مع الشاشات المختلفة.
- `app.js` - منطق التطبيق، معالجة البيانات، الحسابات والتحقق، وتخزين البيانات محلياً.
